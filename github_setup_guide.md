# 초보자를 위한 깃허브(GitHub) 연결 가이드

이 가이드는 재고 관리 데이터를 안전하게 인터넷 공간(GitHub)에 보관하고, 여러 컴퓨터에서 함께 쓸 수 있도록 설정하는 방법을 아주 쉽게 설명합니다.

---

## 1. 깃허브(GitHub)란 무엇인가요?
*   **설명**: 내 컴퓨터에 있는 소중한 데이터(`data.json`)를 인터넷상의 **비밀 창고**에 저장하는 서비스입니다. 
*   **장점**: 컴퓨터가 고장 나도 데이터가 안전하고, 나중에 다른 컴퓨터에서도 똑같이 불러와서 작업할 수 있습니다.

---

## 2. 사전 준비: 깃(Git) 설치하기
현재 사용자님의 컴퓨터에는 '깃(Git)'이라는 프로그램이 설치되어 있지 않습니다. 이 프로그램이 있어야 컴퓨터와 깃허브 창고를 연결할 수 있습니다.

1.  **다운로드**: [Git 공식 홈페이지](https://git-scm.com/download/win)에 접속합니다.
2.  **설치**: 다운로드된 파일을 실행하고, **계속 [Next] 버튼**을 눌러서 설치를 완료해 주세요. (기본 설정 그대로 설치하시면 됩니다.)
3.  **확인**: 설치가 끝나면 저에게 **"설치 완료했습니다"**라고 말씀해 주세요.

---

## 3. 깃허브 계정 만들기
1.  [GitHub.com](https://github.com/)에 접속하여 회원가입을 합니다.
2.  로그인한 뒤, 오른쪽 상단의 **[+]** 버튼을 눌러 **[New repository]** (새 저장소)를 만듭니다.
    *   **Repository name**: `현대재고관리` (영문으로 `hyundai-inventory` 권장)
    *   **Public/Private**: 반드시 **[Private]** (비공개)를 선택하세요! (데이터 보안을 위해 중요합니다.)
    *   맨 아래 **[Create repository]** 버튼을 누릅니다.

---

## 4. 내 컴퓨터와 연결하기 (설치 후 제가 도와드릴 부분)
설치와 계정 생성이 끝나면, 제가 터미널 명령어를 통해 아래 작업들을 대신 처리해 드릴 거예요.

1.  내 폴더를 '깃 관리 구역'으로 지정하기
2.  현재 파일들(`index.html`, `data.json` 등)을 첫 번째 꾸러미로 만들기
3.  깃허브 창고 주소 연결하기
4.  창고로 데이터 밀어넣기 (Push)

> [!NOTE]
> 깃 설치가 완료되어야 제가 다음 명령어를 입력해 드릴 수 있습니다. 설치가 끝나면 꼭 알려주세요!
